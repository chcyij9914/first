기능 

일정
- 일정 목록 보기 기능
- 일정 등록 하기
- 일정 수정 기능
- 일정 삭제 기능
- 일정 색상 지정 기능
- 일정 시간/위치 설정 기능
- 일정 검색 기능
- API 사용 (일정)

캘린더
- 일정과 연동 기능 
- 캘린더 보여주기 기능
- 캘린더 색상 필터 기능 
- API 사용 (캘린더)

리마인더
- 리마인더 문자 전송 기능
- 리마인더 목록 보기 기능
- 홍보용 사진 첨부 기능 
- 홍보용 문자 작성 기능
- API 사용 (SMS)

그래프
- 그래프 구현 기능 (영업실적, 이익분석, 거래건수)
- API 사용 (그래프)

문서
- 문서 목록 조회(검색) 기능
- 문서 등록 기능 
- 문서 PDF 등록 기능
- API 사용 (미정 혹은 잘 모르겠음)

- 영업바에서 일정을 클릭시 보이는 화면은 각 일정이 적혀있는 캘린더와 그날 일정이 작성되어 있는 일정바 
그리고 일정 목록 버튼이 보임 
- 캘린더는 기업에서 내려오는 일정을 빨간색으로 직원이 각자 작성한 일정은 파란색으로 구분되어 캘린더에 작성
- 일정 목록 버튼을 누르면 일정 목록이 있는 창으로 이동
- 일정 목록 화면에는 일정 목록과 일정 등록 버튼, 일정 삭제 버튼 구성
- 일정 등록 버튼을 누르면 일정 제목, 시작날짜_시간(datetime), 종료날짜_시간(datetime) 캘린더 색상(radio), 위치를 기입할 수 있는 텍스트칸 구성
- 일정 목록에서 체크 박스를 통해 체크 후 일정 삭제 버튼을 누르면 일정을 삭제 
- 일정 목록에는 일정 등록에서 작성한 내용들이 보이게 구성
- 대시보드 버튼을 클릭하면 이익분석, 부서별 영업 실적, 거래건수를 보여주는 그래프 구현 
(그래프같은 경우에는 입고 출고 재고 테이블을 통해 나오는 영업 이익 혹은 실적으로 그래프 구현할 계획)
- 리마인더 페이지는 거래 내역 목록과 문자 작성 버튼을 구성 
- 거래내역에서는 체크 박스를 통해 거래처를 선택할수있으며 거래처를 선택후 문자 작성을 누르면 홍보용 SMS 화면이 나옴
- 홍보용 SMS 화면은 홍보용 사진과 홍보용 글귀를 작성할수있으며 전송하기 버튼으로 최종적으로 전송이 가능함 
- 문서 페이지는 세금 계산서 와 영수증의 글귀를 누르면 화면이동함
- 세금 계산서로 이동하면 목록화면에 계산서번호, 발행일자,거래처,과세구분,권,호, 출력횟수, 영수/청구의 내용이 나옴 
- 영수증으로 이동하면 목록화면에 거래일련 번호, 거래일시, 품명, 회사명, 사업자등록번호, 금액 합계등이 나옴
- 각 페이지마다 문서 등록 버튼이 존재하는데 버튼을 누르면 관련 PDF를 등록할 수 있으며, 각 목록화면에서 보여주는 내용들을 직접 작성할 수 있도록 함
